# Vinodkumar's Developer Portfolio

This is my personal developer portfolio built using React, Vite, and Tailwind CSS.  
It showcases my projects, skills, and learning journey as a B.Tech student preparing for placements.

---

##  Features

- Modern and responsive UI
- Projects section with descriptions
- Technical skills with proficiency levels
- Smooth animations and clean design
- Contact and social media integration

---

##  Tech Stack

- React
- Vite
- Tailwind CSS
- ShadCN UI
- TypeScript
